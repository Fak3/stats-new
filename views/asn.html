    An Autonomous System Number ("ASN") describes a unique number assigned to each so-called Autonomous System. A Autonomous System is a group of IP networks with a clearly defined routing policy. Usually an AS coincides with the concept of an Internet Service Provider (ISP). However it must be noted that nowadays, many (larger) ISPs might even have multiple ASNs.
